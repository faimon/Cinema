# Проект "Кинотеатр"

### Web приложение для покупки билетов в кинотеатр.
**Используемые технологии:** Java 14, Java EE, PostgreSQL, Servlets, JSP, JSTL, SQL, JDBC, Maven, Apache Tomcat, Junit, Mockito, HTML, CSS, JS, Ajax.

**Функциональность:**
* Выбор места в зале.
* Покупка билета.
